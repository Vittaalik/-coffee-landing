# Coffee House Landing Page

## Project
One-page landing for a coffee shop. Stack: HTML + CSS + Node.js (Express).

## Files
- server.js - Express server, serves static files from public/
- public/index.html - main page
- public/styles.css - all styles

## Sections
1. Hero - name "Coffee House", subtitle, button
2. About - short text about the coffee shop
3. Menu - 4 coffee items with prices
4. Contact - address, phone, hours

## Design
- Main color: #6F4E37 (coffee brown)
- Background: #FFF8F0 (cream)
- Accent: #D4A574
- Font: Georgia, serif
- Container: max-width 960px, centered

## Important
- server.js serves static files from public/ folder
- index.html links styles.css via link tag
- Server port: 3000
